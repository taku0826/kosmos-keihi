# コスモス経費メモ帳 🌸

**登録不要・完全無料・データはどこにも送信されない**フリーランス向け経費記録・集計ツールです。

👉 **今すぐ使う:** https://taku0826.github.io/kosmos-keihi/

## 特徴

- 📝 **かんたん入力** — 日付・カテゴリ・金額・メモを入力して追加するだけ
- 📊 **カテゴリ別バーグラフ** — どこに使いすぎているかを一目で把握
- 📅 **月ごとの絞り込み** — 先月分だけ、上半期まとめて、など自由に切替
- 📄 **CSVで書き出し** — ExcelやfreeeなどへBOM付きUTF-8で出力。文字化けなし
- 🔒 **完全ローカル動作** — サーバーに一切データを送信しません。入力内容はあなたのブラウザのlocalStorageだけ
- 🧾 **確定申告の科目に対応** — 消耗品費・通信費・交通費・接待交際費・地代家賃など10カテゴリ

## 使い方

1. ページを開く
2. 日付・カテゴリ・金額・メモを入力して「追加する」を押す
3. 月末に「CSVで書き出し」でバックアップ → 確定申告前に会計ソフトへ取り込む

アカウント登録も、アプリのインストールも要りません。

## 注意事項

ブラウザの「閲覧データをすべて削除」操作を行うと、localStorageのデータも消えることがあります。定期的にCSV書き出しでバックアップを取ることをお勧めします。

## 仕組み

単一のHTMLファイルだけで動作します。ビルドなし、依存ライブラリなし。
`index.html` をダウンロードしてオフラインで使うこともできます。
